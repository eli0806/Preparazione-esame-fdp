Il concetto di informazione è astratto, ma è possibile rappresentarlo ed è necessario al fine della manipolazione della stessa.

Per rappresentare un'informazione usiamo un **linguaggio di programmazione** che ci permette di tradurre un algoritmo e poi di interagire con il calcolatore.
A permettere la traduzione del programma ad un linguaggio comprensibile alla macchina è un software chiamato **compilatore**.

L'informazione a livello macchina è rappresentata attraverso una o più sequenze di **bit.**

Il bit o Binary digIT è  è l'unità di misura elementare dell'informazione e base del sistema numerico utilizzato dai computer. Esso può assumere solo 2 valori ( che realisticamente sarebbero equiparabili alla presenza o assenza di un impulso elettrico ) : 0 ( ossia falso, assenza di impulso) oppure 1 (ossia vero, presenza di impulso).

8 bit formano 1 Byte.

L'informazione che una sequenza di bit può esprimere dipende strettamente dalla formula 2^n
dove n è il numero di bit che abbiamo a disposizione. Usiamo la base 2 perché, appunto, abbiamo a disposizione 2 possibilità per bit .

**Rappresentazione testuale**:

Per rappresentare un testo si usano delle convenzioni.
Una di queste è la codifica ASCII (American Standard Code for Information Interchange).
Essa associa al valore di ogni sequenza di 7 bit un carattere (che può essere numerico o alfabetico) o un operazione particolare della macchina.
Ne esiste una versione estesa che sfrutta un intero byte.
La convenzione però più utilizzata ( che possiamo trovare anche quando andiamo a definire pagine HTML ) è la UTF-8 di UNICODE.
